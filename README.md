## Идея компании
Мы создаём сеть кофеен полезного питания «VitaFlow», ориентированную на людей, ведущих здоровый образ жизни: без сахара, низкокалорийное, с акцентом на белок и клетчатку — вкусное ЗОЖ-меню без компромиссов. Мы продаём готовые завтраки, обеды, смузи и десерты клиентам 25–45 лет, которые следят за фигурой, занимаются спортом или хотят питаться правильно вне дома. Ключевой канал продаж — собственные кофейни при фитнес-клубах и отдельные точки в деловых кварталах с интеграцией данных о тренировках клиентов для персонализации рациона.
